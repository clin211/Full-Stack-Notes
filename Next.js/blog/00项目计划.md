## 技术栈

- 数据库 MySQL
- ORM：Drizzle
- 身份认证：Next-Auth
- 状态管理：zustand
- 国际化：next-i18next
- 数据校验：zod
- 环境变量 .env（参考 T3）
- 表单处理 React Hook Form
- 代码质量
    - Husky 用于添加 Git Hooks
    - Prettier 用于代码美化
    - ESlint 用于代码校验
    - lint staged 用于在 git 暂存文件上指定 Lint 检查
    - commitlint 用于校验 Git Commit Messge
    - commitizen 辅助填写提交信息
    - semantic-release 自动化 changelog 生成 自动化包的发布工作流程
- CSS 方案 tailwindcss + shadcn/ui
- 数据分析
    - Pino.js
    - Better Stack
